uint32_t RTE::Physics::PhysicsManager::_maxSubSteps = 10
private

Maximum number of substeps.

In the case that time since last frame is larger than #_fixedTimeStep then bullet will split it into multiple substeps. This value represents the maximum number of sub steps it will be split into.
